📌 Sobre o Projeto
O TelePronto é um aplicativo de atendimento médico remoto criado com o objetivo de reduzir filas em pronto-socorros físicos, permitindo que os pacientes realizem triagem, consultas online e gerenciamento de receitas digitais diretamente pelo celular.

O projeto foi desenvolvido como um protótipo de baixa fidelidade utilizando wireframes simples e com foco em acessibilidade, visual claro e experiência do usuário.

🎯 Objetivo do Aplicativo
O aplicativo busca oferecer:

Triagem rápida de touro
Consulta médica por vídeo
Sala de espera virtual
Gerenciamento de receitas digitais

Acesso rápido a medicamentos e exames
